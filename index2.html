開発環境もろもろ

ＸＮＡ4.0 を使用したゲームが動作する環境

  • OS は Windows XP SP3 以降、Windows Vista SP1 以降、Windows 7以降
  • シェーダモデルバージョン 2.0 以上に対応したビデオカード、またはチップセット013/09/08 ver1.0
  • ゲームの内容に応じた CPU, メモリ, HDD, ビデオカード
    これらの条件を満たした上で、以下のコンポーネントもインストールする必要があります。
  • Microsoft.NET Framework 4 （大抵の環境では既にインストールされていると思う）
  • Microsoft XNA Framework Redistributable 4.0

F#言語を使用したゲームをプレイするには

  • 下記のランタイムをインストールする必要があります。
  • Microsoft Visual Studio 2010 F# Runtime 2.0
  • Microsoft Visual Studio 2010 F# 日本語 Language Pack（こっちは要らないかも）
    ※F#3.0を使用しているゲームに関しては、ライブラリを同梱しているためランタイムは必要ないはずです。

開発環境(プロジェクトファイルを開くのに必要なツール)※すべて無償

  • No.1～12のゲーム（HSP）
    • 開発環境：HSP
  • No.14～27のゲーム(F#2.0)
    • 統合開発環境：Microsoft Visual Studio 2010 Shell (Integrated)
      ※インストールするとこっちも入れろと言われる→Microsoft Visual Studio 2010 Service Pack 1
    • F#2.0のコンパイラとツール：Microsoft F#, August 2010 Community Technology Preview
    • XNA4.0の開発者向けライブラリとツール：Microsoft XNA Game Studio 4.0
      ※No.21以降のゲームでは不要
  • No.28,29 のゲーム(F#3.0)
    • ここに書いてある通りインストールすれば、開発環境がインストールできる。
      ※おそらくF#2.0のプロジェクトもこちらで開けると思います。
  • No.30～ （C# + WPF）
    • 統合開発環境：Visual Studio Express 2012 for Windows Desktop
  • No.37～ (Java + AndroidSdk + AndEngine)
    • 統合開発環境：Eclipse(AndroidSdk付属)
    • ライブラリ：AndEngine

愛用のおすすめツール

  • HSP ： 管理人の使用しているプログラム言語です。ドキュメントが親切、サンプルが多い、情報が多い、ゲーム作成に特化した機能の豊富さが魅力。
  • JoyToKey ： ジョイパッドに対応していないゲームがジョイパッドで遊べるツールです。当サイトのゲームでももちろん使用できます。
